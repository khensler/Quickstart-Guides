## Quick Reference

| Command | Description |
|---------|-------------|
| `sudo nvme list-subsys` | Show all paths and subsystems |
| `sudo nvme list` | List NVMe devices |
| `sudo nvme connect -t tcp -a <IP> -s 4420 -n <NQN>` | Connect to subsystem |
| `sudo nvme disconnect -n <NQN>` | Disconnect from subsystem |
